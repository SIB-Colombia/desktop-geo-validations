Desktop-geo-validations

Validations of geographical coordinates from files.

Download .zip Download .tar.gz View on GitHub

Introducción

Esta aplicación es útil para verificar coordenadas geoespaciales a través de archivos .txt. Por el momento esta validación esta disponible solo para:

  • Municipios
  • Áreas Marítimas

Estructura principal del proyecto

La aplicación tiene los siguientes archivos:

  • client_config_areas.txt
  • client_config_municipios.txt
  • Validar.bat (Windows)
  • Validar.sh (Linus/Mac) y dos carpetas:
  • AreasMaritimas
  • Municipios

Cada carpeta contiene información sobre:

  • input: En esta carpeta contiene un .txt con la información que debe ser verificada.
  • output: En esta carpeta se encuentra el .txt con la información verificada.
  • shapefile: Esta carpeta contiene varios archivos, los cuales contienen la información geografica necesaria para llevar acabo la verificación.
  • SHPMunicipioV.jar o SHPAreaMaritimaV.jar

Nota

Puede descargar:

  • Shapefile: https://s3.amazonaws.com/IPT/shapes/LimitesMaritimos.zip and https://s3.amazonaws.com/IPT/shapes/Departamentos(2012).zip
  • Java ARchive: https://github.com/SIB-Colombia/desktop-geo-validations/tree/gh-pages/Java%20ARchive

Estructura de los archivos .txt

Archivo de entrada txt

Para el caso de Áreas Marítimas:

El archivo txt debe llamarse "areaMaritimas.txt" y necesita tener la siguiente información en el mismo orden que se da:

  • id
  • latitud
  • longitud
  • Nombre del Área Marítima

Para el caso de Municipios:

El archivo txt debe llamarse "municipios.txt" y necesita tener la siguiente información en el mismo orden que se da:

  • id
  • latitud
  • longitud
  • Nombre del Municipio
  • Nombre del Departamento

Archivo de salida txt

Para el caso de Áreas Marítimas:

El archivo txt que se crea muestra la información en el siguiente orden:

  • id
  • latitud
  • longitud
  • Nombre del Área Marítima
  • Nombre del Área Marítima calculada
  • True or False (verificación del área marítima)

Para el caso de Municipios:

El archivo txt que se crea muestra la información en el siguiente orden:

  • id
  • latitude
  • longitude
  • Nombre del Departamento
  • Nombre del Departamento calculado
  • True or False (verificación del departamento)
  • Nombre del Municipio
  • Nombre del Municipio calculado
  • True or False (verificación del municipio)
